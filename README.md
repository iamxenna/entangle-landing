# Entangle Landing Project

## Development

### Framework: Next.JS with Preact

### Styles: Styled-Components

---

## Available Scripts

#### `yarn install / yarn`

Installing dependencies

#### `yarn start`

After build, starts the server on port 3000

#### `yarn dev`

Project launch on - [http://localhost:3000](http://localhost:3000)

#### `yarn test`

Start testing

#### `yarn build`

Creating a build
